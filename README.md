# SQL.-Hospital-Patient-Records-Management-System
Project Overview The Hospital Patient Records Management System is a database project designed to  manage patient information, medical history, and appointments in a hospital environment. Many hospitals still rely on paper-based or poorly organized systems, which leads to errors,  data loss, and slow access to patient records.

This project focuses on designing and implementing a relational database using MySQL to 
securely store and manage patient records in an efficient and structured way.

Tools Used
- MySQL
- SQL (Structured Query Language)
- MySQL Workbench
- ER Diagram Tool (Draw.io / Lucidchart)
- GitHub (project documentation and version control)
